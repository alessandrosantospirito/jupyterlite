Function

Input Parameters

  • threshold∈[0,1], with default value 0.

distributions=[d1⋮dm]=[[[μ11,μ21],[σ001,σ011],[σ101,σ111]]⋮[[μ1m,μ2m],[σ00m,σ01m],[σ10m,σ11m]]]∈Mm×3×2,points=[p1⋮pn]=[[px1,py1]⋮[pxn,pyn]]∈Mn×2.

Methodology

Let s be defined as s:=min(n,m). We will calculate the likelihood values for each point and find the s points for each distribution that have the highest likelihood.

  • For matching, it is sufficient to only consider the top s point matches.

  • We define p~i,j∈{p1,…,pn}:∑k=1ilik(dj,p~k,j)≥∑k=1ilik(dj,p^k) for every ordered subset P^⊆P.

likelihoods=[lik(d1,p~1,1)…lik(dm,p~1,m)⋮lik(d1,p~s,1)…lik(dm,p~s,m)]∈Ms×m.

Output

For the output, we make a small adjustment to the likelihood. We define

likelihood entries=[lik~(d1,p~1,1)…lik~(dm,p~1,m)⋮lik~(d1,p~r,1)…lik~(dm,p~r,m)]∈Mr×m,lik~(di,pj):={lik(di,pj),if lik(di,pj)≥threshold10−50,otherwiselikelihood indices to filter=[bool-lik(d1,p~1,1)…⋮bool-lik(d1,p~r,1)…]∈Mr×m,bool-lik(di,pj):={False,if lik(di,pj)≥thresholdTrue,otherwise

Here, r is the count of all points that were in at least one of the "top s likelihoods" of any distribution:

r:=|{p~i,j∣1≤i≤s,1≤j≤m}|,s≤r≤m2.
